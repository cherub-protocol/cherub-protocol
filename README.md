# Cherub

Inverse Perpetuals (3, 3) DAO

## Development

**Requirements**

- Anchor 0.16.2
- Solana 1.8.0
- Node 14.18.0

**Running**

Make sure cluster is set to either `localnet` or `devnet` in `Anchor.toml`, depending on the command.

```bash
yarn test:localnet
```

Start the frontend app.

```bash
yarn start:app
```

## Documentation

This is currently handled with Docusauraus.

```bash
yarn start:docs
```
